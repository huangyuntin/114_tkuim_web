自訂錯誤條件

- 姓名：必填
- Email：必填、Email 格式錯誤
- 手機：必填、必須剛好是 10 碼數字
- 密碼：必填、至少 8 碼、必須為英數混合
- 確認密碼：必填、必須與密碼欄位一致
- 興趣標籤：必填（至少勾選 1 個）
- 服務條款：必填（必須勾選）

加分項功能

1.  顯示密碼強度條：
    在 password 欄位 input 時即時觸發
    根據「長度、數字、小寫、大寫、特殊符號」的組合規則，將強度分為「弱 (紅)、中 (橘)、強 (綠)」，並更新 CSS 寬度與顏色

2.  localStorage 暫存：
    在表單任何欄位 input 時，會自動將當前所有（未送出）的資料（包含複選框）序列化為 JSON，並儲存至  localStorage 
    當使用者重新整理或關閉分頁後再次開啟， DOMContentLoaded 事件會觸發  loadFormData() ，自動從  localStorage 讀取並恢復資料
    表單成功送出後，會自動清除此暫存

3.  實作「重設」按鈕：
    點擊「重設」按鈕會觸發  resetFormState()  函式
    此功能不僅會清除欄位內容 ( form.reset() ) 還會：
        清除所有顯示的錯誤訊息
        移除所有  .validated ,  .invalid ,  .valid 狀態 class
        重設密碼強度條
        清除  localStorage 暫存資料
        將焦點移回第一個欄位（姓名）

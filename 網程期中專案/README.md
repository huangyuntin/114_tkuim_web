# 網程專案名稱：阡谷室內設計服務預約與估價網頁

組長 : 范語仁
組員 : 黃韻庭

# 專案簡介
本專案旨在為「阡谷室內空間設計」製作一個具備實際互動功能的線上服務預約與估價系統。系統整合了 HTML、Bootstrap 5 與原生 JavaScript，讓使用者能即時選擇所需的設計與施工服務，並根據輸入的坪數自動計算預估總價。系統亦實作了嚴謹的表單驗證機制，以提升使用者體驗與資料準確性。

# 使用技術
HTML5：語意化結構設計
CSS3 / Bootstrap 5：採用 RWD 響應式設計，確保在不同裝置上顯示正常。
JavaScript (ES6)：核心互動功能邏輯。
DOM 操作：用於即時更新價格與顯示成功訊息。
HTML5 Constraint Validation API：配合自訂驗證提示。

# 功能特色

1. 互動功能：即時服務估價計算
使用者在「預約與估價」區塊選擇服務項目（設計、施工、監工）及輸入坪數時，系統會即時計算並顯示預估總價（單價乘以坪數總和）。
此功能透過 JavaScript 監聽輸入欄位和勾選框的 `input` / `change` 事件來實現。

2. 表單驗證與回饋
標準驗證：對姓名、Email、電話、坪數等必填欄位使用 HTML5 `required` 屬性及 `type` 屬性進行基礎格式檢查。
自訂驗證：使用 JavaScript 額外檢查坪數是否為有效數字且大於最小限制（例如 5 坪），以及確保使用者至少選擇一項服務項目。
使用體驗：驗證失敗時，會觸發 Bootstrap 的錯誤樣式提示並顯示彈跳視窗提醒。

# 3. 互動功能：提交狀態回饋
表單送出成功後，透過 DOM 操作動態隱藏表單內容，並在頁面底部顯示一則包含使用者姓名的感謝成功訊息，提供視覺上的狀態確認。
